# 支付失败修复报告 (400 Bad Request)

已为您修复支付失败的问题。

## 问题原因
根据您的报错日志 `{"billId":4,"payMethod":"wx","amount":"171"}` 可以看出，`amount` 字段是字符串类型 (`"171"`)。而后端接口（Java BigDecimal 或 Double）通常需要数字类型。当发送字符串时，后端 JSON 反序列化可能会失败，或者参数校验不通过，导致返回 400 错误。

## 修复内容
在 `owner/pages/payment/pay.vue` 中，在发送支付请求前，显式将 `amount` 转换为数字：

```javascript
// 修改前
amount: this.amount

// 修改后
amount: Number(this.amount)
```

现在请求体将变为 `{"billId":4, "payMethod":"wx", "amount": 171}`，这应该能被后端正确解析。

## 验证方法
请再次尝试支付流程，现在应该可以支付成功了。
