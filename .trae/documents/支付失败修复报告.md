# 支付失败修复报告

已为您修复支付失败 (400 Bad Request) 的问题。

## 问题原因
后端接口要求 `billId` 必须是数字类型（例如 Long 或 Integer），但前端从页面路由参数 (`options.billId`) 获取到的值默认为**字符串**类型。当直接将字符串类型的 `billId` 发送给后端时，后端在 JSON 反序列化或参数校验时失败，返回了 400 错误。

## 修复内容
在 `owner/pages/payment/pay.vue` 中，在发起支付请求前，显式地将 `billId` 转换为整数：

```javascript
// 修改前
billId: this.billId

// 修改后
billId: parseInt(this.billId)
```

## 验证方法
请重新尝试支付流程：
1. 进入“物业缴费”页面。
2. 点击“待缴费”列表中的任意账单。
3. 在收银台点击“确认支付”。
4. 应该能看到“支付成功”的提示，并自动返回上一页。
